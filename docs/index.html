IP-Locator
What's This?

This site is using an AWS Lambda API as an intermediary between you and ip-api.com.
The Lambda function captures your IP address, gets details from ip-api, then serves the info back the format requested. This API can gather info for any valid IP address, and can provide either a JSON response or a simplified HTML response. See the respository for full details.
